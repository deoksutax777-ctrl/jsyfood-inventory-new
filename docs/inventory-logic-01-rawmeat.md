# 원육 수불부 계산 로직 (검증 완료)

## 1. 전체 구조

```
월별 시트 × 7개 섹션:
  전사 = 육가공 (동일, 합산 롤업)
  컨테이너 (매입 수령 → 업체 배분)
  참프레시 / 영미트 / 인천공장 / 스마일푸드 (개별 업체)

전사 = 참프레시 + 영미트 + 인천공장 + 스마일푸드 (단순 합산)
```

## 3. 계산 순서 (월별)

### Step 1: 기초재고 설정
```
당월 기초 = 전월 기말
(1월은 전년도 12월 기말 수동 입력)
```

### Step 2: 컨테이너 입고 ← 매입자료
```
컨테이너.입고수량 = 매입자료 해당월 품목별 합계 수량
컨테이너.입고단가 = 매입자료 해당월 품목별 가중평균단가
컨테이너.입고금액 = 매입자료 해당월 품목별 합계 금액
컨테이너.기초 = 항상 0 (즉시 배분)
컨테이너.기말 = 항상 0
컨테이너.출고 = 컨테이너.입고 (전량 배분)
```

### Step 3: 업체별 입고 ← 컨테이너 출고 배분
```
각 업체.입고수량 = 작업일지 해당업체 해당월 품목별 입고수량 합계
각 업체.입고단가 = 컨테이너.입고단가 (매입단가 그대로 전달)
각 업체.입고금액 = 입고수량 × 입고단가

검증: Σ(모든업체.입고수량) = 컨테이너.출고수량
```

### Step 4: 업체별 출고 ← 작업일지
```
각 업체.출고수량 = 작업일지 해당업체 해당월 품목별 출고수량 합계
```

### Step 5: 총평균법 단가 계산 (핵심)
```
총평균단가 = (기초금액 + 입고금액) / (기초수량 + 입고수량)
출고단가 = 총평균단가
출고금액 = 출고수량 × 총평균단가
기말수량 = 기초수량 + 입고수량 - 출고수량
기말금액 = (기초금액 + 입고금액) - 출고금액
기말단가 = 기말금액 / 기말수량  (= 총평균단가와 동일)
```

### Step 6: 전사/육���공 합산
```
전사.기초 = Σ(각 업체.기초)
전사.입고 = Σ(각 업체.입고) = 컨테이너.입고
전사.출고 = Σ(각 업체.출고)
전사.기말 = Σ(각 업체.기말)
전사.단가 = 전사.금액 / 전사.수량 (가중평균)
```

## 4. 특수 처리

### 우사각 (살치우사각 / 부채우사각)
- 육가공업체에서 작업용으로 별도 출고한 수량
- 전사 입고에 우사각 입고 가산, 전사 출고에 우사각 출고 가산
- 26년 작업일지에서 해당 항목 존재 여부 확인 필요

### 입고=0, 단가 유지 케이스
- 입고수량=0이면 입고단가=0, 총평균단가=기초단가 그대로 유지
- 기초=0, 입고=0이면 해당 품목 스킵
