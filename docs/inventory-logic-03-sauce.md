## 5. 계산 순서 (월별)

### Step 1: 원재료 입고 ← 매입자료
```
원재료.입고수량 = 매입자료 해당월 품목별 합계 수량
원재료.입고단가 = 매입자료 해당월 품목별 가중평균단가
원재료.입고금액 = 매입자료 해당월 품목별 합계 금액
```

### Step 2: 원재료 총평균법
```
총평균단가 = (기초금액 + 입고금액) / (기초수량 + 입고수량)
출고단가 = 총평균단가
출고금액 = 출고수량 × 총평균단가
기말 = 기초 + 입고 - 출고
```

### Step 3: BOM 비례배분 → 제품 입고 (가공육과 동일 로직!)

**3-1. 표준무게합 산출**
```
표준무게합(제품, 원재료) = 생산수량(제품) × 표준무게(제품, 원재료)
전체표준무게합계(원재료) = Σ 표준무게합(모든 제품, 해당 원재료)
```

**3-2. 원재료사용비율 계산**
```
원재료사용비율(제품, 원재료) = 표준무게합(제품, 원재료) / 전체표준무게합계(원재료)
```

**3-3. 투입원가 배분**
```
투입원가(제품, 원재료) = 원재료사용비율(제품, 원재료) × 원재료출고금액(원재료)
```

**3-4. 제품 입고 확정**
```
제품 입고수량 = 생산수량
제품 입고금액 = Σ 투입원가(해당 제품, 모든 사용 원재료)
제품 입고단가 = 입고금액 / 입고수량
```

### Step 4: 제품 출고 (총평균법)
```
출고단가 = (기초금액 + 입고금액) / (기초수량 + 입고수량)
출고금액 = 출고수량 × 총평균단가
기말 = 기초 + 입고 - 출고
```

### Step 5: 공장간 이동
```
인천공장 출고 → 참프레시/영미트 입고
이동단가 = 인천공장 총평균단가
```

## 7. 가공육과의 차이점

| 항목 | 원육→가공육 | 원재료→소스 |
|-----|-----------|-----------|
| 배분 원천 | 원육 수불부 출고금액 | 원재료 출고금액 (동일 시트 내) |
| 배분 공식 | **동일** (BOM 비례배분) | **동일** |
| 생산 공장 | 참프레시/영미트/인천 | 인천공장만 |
| 제품 수 | ~15종 | 2종 |
| BOM 변동 | 월별 변동 가능 | 고정 (11~12월 동일) |
| 원재료 입고 | 원육 수불부에서 독립 관리 | 동일 시트 내 원재료 섹션 |

## 8. 데이터 흐름

```
[매입자료] → 원재료 입고 (11종)
     │
     ▼
원재료 총평균법 → 원재료 출고단가/출고금액
     │
     ▼
[BOM 표준무게 테이블] + [생산수량]
     │
     ▼
원재료사용비율 = (생산수량 × 표준무게) / Σ(전체 표준무게합)
     │
     ▼
투입원가(제품) = Σ( 비율 × 원재료출고금액 )  ← 제품 입고금액
     │
     ▼
제품 총평균법 → 출고단가
     │
     ▼
인천 → 참프레시/영미트 이동 (인천 총평균단가로)
```
