## 3. 핵심 발견: 가공비는 입고원가에 미포함

```
가공육 입고금액 = 원육 출고금액의 BOM 비례배분 (가공비 제외!)
가공비는 별도 손익계산에만 반영 (수율 시트: 손익 = 매출 - 원가 - 가공비)

검증: 가공육 전사 입고금액 합계 = 원육 전사 출고금액 합계 (원-원 일치)
```

## 4. 계산 순서 (월별)

### Step 1: 기초재고
```
당월 기초 = 전월 기말
```

### Step 2: BOM 비례배분 → 가공육 입고

**2-1. 각 원육 부위별로 표준무게합계 산출**
```
표준무게합(품목, 원육) = 생산수량(품목) × 표준무게(품목, 원육)
전체표준무게합계(원육) = Σ 표준무게합(모든 품목, 해당 원육)
```

**2-2. 부위별표준사용비율 계산**
```
부위별표준사용비율(품목, 원육) = 표준무게합(품목, 원육) / 전체표준무게합계(원육)
```
- 생산수량=0인 품목 → 표준무게합=0 → 비율=0 (자동 제외)
- 각 원육별 비율 합계 = 1.0

**2-3. 투입원가 배분**
```
투입원가(품목, 원육) = 부위별표준사용비율(품목, 원육) × 원육출고금액(원육)
```
- 원육출고금액 = 원육 수불부의 해당 업체 해당 원육 출고금액

**2-4. 가공육 입고 확정**
```
가공육 입고수량(품목) = 생산수량 (작업일지에서)
가공육 입고금액(품목) = Σ 투입원가(품목, 모든 사용 원육)
가공육 입고단가(품목) = 입고금액 / 입고수량
```

### Step 3: 출고 ← 작업일지
```
각 업체.출고수량 = 작업일지 출고 합계 (매장+본푸드+샤인+기타)
출고단가 = 총평균단가
출고금액 = 출고수량 × 총평균단가
```

### Step 4: 총평균법 (원육과 동일)
```
총평균단가 = (기초금액 + 입고금액) / (기초수량 + 입고수량)
기말수량 = 기초수량 + 입고수량 - 출고수량
기말금액 = (기초금액 + 입고금액) - 출고금액
기말단가 = 기말금액 / 기말수량 (= 총평균단가)
```

### Step 5: 공장간 이동 처리
```
이동 = from업체 출고 + to업체 입고
이동단가 = from업체의 총평균단가
전사 합계에는 영향 없음 (내부 상계)
```

### Step 6: 전사 합산
```
전사 = Σ(각 업체) - 내부이동 중복 제거
```

## 6. 특수 품목 처리

### 안창살/우설 (원육 소분 판매)
```
표준무게=1.0, 사용비율=1.0, 부위별비율=1.0
입고수량 = 원육출고수량 (그대로)
입고금액 = 원육출고금액 (그대로)
입고단가 = 원육출고단가 (그대로)
```

### 깐양 (막내장+깐양 혼합)
```
막내장: 사용비율 0.5, 표준무게 0.5 → 부위별비율 1.0
깐양:  사용비율 0.5, 표준무게 0.5 → 부위별비율 1.0
입고금액 = 막내장출고금액 + 깐양출고금액
생산수량 = 막내장원육수량 × 2 (= 깐양원육수량 × 2)
```

### 돼지갈비(동그랑땡) (공장간이동 포함)
```
돈목살(1.8kg/pack) + 돼지뼈(0.6kg/pack)로 생산
인천공장에서 주로 생산 → 참프레시/영미트로 공장간이동
이동받는 업체: BOM 투입원가=0, 입고금액은 이동단가로 별도 반영
```

### 입고=0 케이스
```
해당 원육의 원육수량=0이면 투입원가=0
생산은 발생했으나 원가=0으로 기록 (예: 참프레시 돼지갈비 입고270대, 입고금액0)
```

## 8. 데이터 흐름 요약

```
[원육 수불부] ──→ 업체별/원육별 출고금액 (총평균법으로 산출)
     │
     ▼
[BOM 테이블] ──→ 품목별 표준무게/사용비율 (고정 참조)
     │
     ▼
[작업일지] ──→ 품목별 생산수량
     │
     ▼
부위별표준사용비율 = (생산수량 × 표준무게) / Σ(전체 표준무게합)
     │
     ▼
투입원가(품목) = Σ( 부위별비율 × 원육출고금액 )   ← 이것이 가공육 입고금액!
     │
     ▼
[작업일지] ──→ 품목별 출고수량
     │
     ▼
총평균법: 출고단가 = (기초금액 + 입고금액) / (기초수량 + 입고수량)
     │
     ▼
기말재고 = 기초 + 입고 - 출고
```

## 9. 원육 수불부와의 연결

```
원육 수불부의 "업체별 출고금액" = 가공육 BOM 배분의 원천 데이터
  → 원육 참프레시 출고 갈비 금액 = 가공육 참프레시 BOM에서 갈비 투입원가의 총합
  → 완전 일치 (1원 이내)

가공육 전사 입고금액 합계 = 원육 전사 출고금액 합계 (가공비 제외이므로 원-원 매칭)
```
